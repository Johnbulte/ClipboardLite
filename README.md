# 轻剪 Cliply

轻剪 Cliply 是一个轻量、现代化的桌面剪贴板增强工具。它会在本地保存文本复制历史，支持搜索、分类、收藏和一键复用，适合日常写作、开发、客服、运营等高频复制场景。

## 已实现功能

- 自动监听文本剪贴板
- 本地保存复制历史
- 自动识别内容类型：文本、链接、代码、邮箱、颜色
- 搜索历史记录
- 收藏常用内容
- 复制历史内容回剪贴板
- 删除单条记录 / 清空历史
- 托盘菜单
- 全局快捷键 `Ctrl + Shift + V`
- 现代化玻璃拟态界面

## 运行

```bash
npm install
npm start
```

## 项目结构

- `src/main.js`：Electron 主进程、剪贴板监听、数据持久化、托盘和快捷键
- `src/preload.js`：安全 IPC 桥接
- `src/renderer/`：界面与交互
- `02-design/`：原型与设计资料
